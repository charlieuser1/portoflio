Proyecto de estudio.

Propuesta de rediseño para

Para esta propuesta de rediseño apliuqe mis conocimientos de Ux/Ui avanzado con el fin de aumentar la intuitividad del usuario encontrando los puntos de dolor del usuario mediante entrevistas, encuestas y testeos, logrando realizar mediante los insigths reducir la carga pensativa y el tiempo de decision de los usuarios.

Durante el proceso de rediseño opté por realizar un diseño mas amigable y familiar para los usuarios que usan plataformas de anuncios clasificados como esta.

La utilzacion de conceptos simples como las "cards" logran ser muy bien aceptadas por los usuarios. Las "cards" organizan la información en bloques individuales, lo que facilita la lectura y exploración de productos. Cada "card" puede contener elementos esenciales como imágenes, descripciones, precios entre otras funciones.

Es crucial que el usuario pueda leer la información relevante de un producto a primera vista por varias razones, principalmente:

1. Captar la atención rápidamente Los usuarios toman decisiones en cuestión de segundos al explorar una página. Mostrar información clave (como el nombre del producto, precio, y características destacadas) evita que pierdan interés y los guía hacia la acción deseada, como hacer clic o comprar.

2. Mejorar la experiencia del usuario Un diseño claro y accesible facilita que los usuarios encuentren lo que buscan sin frustraciones. Esto hace que la navegación sea más eficiente y placentera, lo que aumenta la probabilidad de que el usuario se quede en la página.

3. Transparencia y confianza Cuando un usuario ve información relevante de inmediato, como precio o beneficios principales, percibe que la marca es transparente. Esto genera confianza, especialmente en productos nuevos o menos conocidos.

4. Tomar decisiones rápidas Proporcionar información clave ayuda a los usuarios a decidir más rápido si el producto cumple con sus necesidades, lo que puede acelerar el proceso de compra.

Proyecto AllwavesWorld.

Proyecto de Desarrollo Web - propuesta AllwavesWorld

Utilicé HTML5 para estructurar el contenido de manera clara y semántica, asegurando una mejor indexación y accesibilidad. Para los estilos, implementé CSS junto con SCSS, lo que me permitió modularizar y optimizar la presentación visual de la web. La disposición de los elementos se organizó utilizando Bootstrap para garantizar un diseño responsive y adaptable a distintos dispositivos. Durante el desarrollo en Visual Studio Code, aproveché sus herramientas para la escritura eficiente de código, con extensiones que facilitaron la gestión de estilos y depuración del código. Además, integré animaciones CSS para mejorar la experiencia del usuario y dar una sensación más fluida al navegar. En conjunto, este diseño busca transmitir la identidad de AllWaveWorld mediante una interfaz limpia y funcional, optimizada para la navegación fluida y atractiva.
