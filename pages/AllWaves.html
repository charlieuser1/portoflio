Proyecto AllwavesWorld.

Proyecto de Desarrollo Web - propuesta AllwavesWorld

Ver repositorio Github
Ver Pagina (github)

Para esta propuesta de desarrollo web utilicé HTML5 para estructurar el contenido de manera clara y semántica, asegurando una mejor indexación y accesibilidad.
Para los estilos, implementé CSS junto con SCSS, lo que me permitió modularizar y optimizar la presentación visual de la web. La disposición de los elementos se organizó utilizando Bootstrap para garantizar un diseño responsive y adaptable a distintos dispositivos.

Durante el desarrollo en Visual Studio Code, aproveché sus herramientas para la escritura eficiente de código, con extensiones que facilitaron la gestión de estilos y depuración del código. Además, integré animaciones CSS para mejorar la experiencia del usuario y dar una sensación más fluida al navegar. En conjunto, este diseño busca transmitir la identidad de AllWaveWorld mediante una interfaz limpia y funcional, optimizada para la navegación fluida y atractiva.
